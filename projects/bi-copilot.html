Overview

This project is a full-stack AI Business Intelligence system that connects Claude AI directly to a live Snowflake data warehouse using Anthropic's Model Context Protocol (MCP). Users can type natural language questions like "What are the top 5 products by revenue?" and receive instant, accurate SQL-backed answers.

The system includes a multi-page Streamlit dashboard for data exploration, a natural language query interface, an AI-powered insights dashboard for anomaly detection, and a fully integrated MCP server that works inside VS Code — allowing Claude to query live business data in real time during conversations.

The architecture uses a factory pattern to seamlessly switch between DuckDB (local development) and Snowflake (production), with Docker for containerized deployment and a comprehensive test suite of 31 automated tests.
