Overview

This project demonstrates how to build and deploy a fully interactive data science application using Streamlit — making complex data analysis accessible to non-technical users through a clean browser interface.

The app allows users to upload datasets, explore data with automated EDA, generate interactive visualizations, and run machine learning models — all without writing a single line of code. It serves as both a practical tool and a showcase of rapid application development skills.

The project laid the foundation for the larger BI Copilot project, establishing the Streamlit-based architecture pattern and data visualization techniques that were later scaled and connected to Snowflake and Claude AI.
